### 5. Backend — CORS Middleware Merkezi Hale Getirildi

**Sorun:** Her handler fonksiyonunda `w.Header().Set("Access-Control-Allow-Origin", "*")` tekrarlanıyordu. Ayrıca `OPTIONS` preflight istekleri hiç karşılanmıyordu. Modern tarayıcılar `PUT`, `DELETE` gibi bazı istekler öncesinde OPTIONS isteği atar; cevap gelmezse asıl isteği göndermez.

**Çözüm:** `main.go`'ya `corsMiddleware` eklendi:
```go
func corsMiddleware(next http.Handler) http.Handler {
    return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        w.Header().Set("Access-Control-Allow-Origin", "*")
        // ...
        if r.Method == http.MethodOptions {
            w.WriteHeader(http.StatusNoContent)
            return
        }
        next.ServeHTTP(w, r)
    })
}
```
Handler'lardan tekrarlı header satırları kaldırıldı.

### 6. Backend — Route Çakışması Düzeltildi

**Sorun:** Go'nun `net/http` mux'unda `/api/physiotherapists/` pattern'i hem `/api/physiotherapists/` hem de `/api/physiotherapists/5` gibi tüm alt path'leri yakalar. Eski kodda `/api/physiotherapists` (liste) ve `/api/physiotherapists/` (detay) ayrı handler'lara bağlıydı ama mux `/api/physiotherapists` isteğini önce liste handler'ına, ardından çakışma olduğunda detay handler'ına da iletebiliyordu.

**Çözüm:** `routes.go` güncellendi — `/api/physiotherapists/` handler'ı içinde path parsing yapılarak ID var mı yok mu kontrol edilir:
```go
mux.HandleFunc("/api/physiotherapists/", func(w http.ResponseWriter, r *http.Request) {
    parts := strings.Split(strings.TrimSuffix(r.URL.Path, "/"), "/")
    if len(parts) == 4 && parts[3] != "" {
        handlers.GetPhysiotherapistByID(w, r)
        return
    }
    handlers.GetPhysiotherapistsByCity(w, r)
})
```

### 7. Handler'larda `http.Error` yerine Structured JSON Hata Yanıtı

**Sorun:** `http.Error(w, '{"error":"..."}', statusCode)` kullanımı yanlış — `http.Error` `Content-Type: text/plain` yazar, bu yüzden frontend `JSON.parse` yaparken hata alır.

**Çözüm:** Tüm hata yanıtları `json.NewEncoder(w).Encode(map[string]string{"error": "..."})` formatına çevrildi ve `w.WriteHeader(statusCode)` öncesinde çağrılır.
