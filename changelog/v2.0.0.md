## 2. 🗺️ Leaflet Harita Entegrasyonu

### Ne Değişti?
`MapSelector.vue` içindeki statik SVG placeholder kaldırıldı. Yerine **Leaflet.js** tabanlı interaktif bir harita eklendi.

### Neden Leaflet?
- Google Maps API ücretlidir; Leaflet + OpenStreetMap tamamen ücretsizdir
- CDN üzerinden çalışır, Vue paketine dahil edilmesine gerek yoktur
- `window.L` aracılığıyla Vue Options API ile kolayca entegre edilir
- Türkiye'nin tüm il merkezleri için koordinat tablosu oluşturuldu

### Nasıl Çalışır?
1. `index.html`'e Leaflet CDN (CSS + JS) eklendi
2. `mounted()` hook'unda `window.L` kontrol edilerek harita `#turkey-map` div'ine bağlandı
3. Her il için koordinat tablosu (`cityCoords`) bileşene eklendi (81 il)
4. Her il, `L.circleMarker` ile mavi daire olarak gösterildi
5. `mouseover` → turuncu renk + büyüme animasyonu
6. `click` → `selectCity()` tetiklenerek `city-selected` eventi emit edildi
7. Seçili şehir → yeşil daire olarak kalır

### Koordinat Kaynağı
Her ilin yaklaşık merkez koordinatı (enlem/boylam) manuel olarak tabloya girildi. Hassas sınır tabanlı SVG harita için `turkey-map` SVG dosyaları veya GeoJSON kullanılabilir (gelecek geliştirme).

## 🔧 Teknik Notlar

### Leaflet ve Vue Uyumu
`window.L` kullanılmasının sebebi: Leaflet, CDN üzerinden global `window` nesnesine yazılır. Vue component içinde `import L from 'leaflet'` yapılamaz çünkü npm paketi yüklü değil. Bu nedenle `mounted()` içinde `window.L` varlığı kontrol edilir.

### Skeleton Loading Neden Önemli?
Backend çağrıları network gecikmesi yaşayabilir. "Yükleniyor..." yerine iskelet kartlar göstermek kullanıcıya içeriğin *geleceğini* hissettirerek bekleme algısını azaltır (Perceived Performance).

### Responsive Tasarım
- `@media (max-width: 640px)` ile mobil kırılım noktaları eklendi
- `clamp()` ile hero başlığı viewport genişliğine göre ölçekleniyor
- Info grid 480px altında 2 sütuna düşüyor
